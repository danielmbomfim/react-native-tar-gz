# @react-native/tar-gz

A react native package for compressing and extracting tar.gz files

## Installation

```sh
npm install @react-native/tar-gz
```

## Usage


```js
import { multiply } from '@react-native/tar-gz';

// ...

const result = multiply(3, 7);
```
